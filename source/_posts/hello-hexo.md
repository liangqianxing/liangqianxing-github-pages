经过几年的“暂住”在 GitHub Pages 自带的模板上，我终于动手把这里换成 Hexo + NexT 的新家。UI、交互和写作体验一下子清爽了很多，也顺手把整个开发流程做了自动化。

<!-- more -->

## 这次升级做了什么

- 🎨 重新设计了首页 Hero、色板和排版，让第一眼就能感受到我的气质。
- ⚙️ 使用 Hexo 8 + NexT 主题，自定义样式放在 `source/_data`，后续升级主题也不会冲突。
- 🚀 配好 GitHub Actions，提交代码就可以自动构建并部署到 Pages。
- 🧰 加了 RSS、Search 和标签 / 分类，内容导航更舒服。

这里收集了我最近维护的几个 side project，它们大多是为了验证一个小想法而诞生，也欢迎你 fork / 写 issue 一起完善。

### 01 · Aurora UI
- **类型**：开源 Design System
- **技术**：Figma Tokens + Vue 3 + UnoCSS
- **亮点**：一套适合 SaaS 产品的多主题组件库，内置 80+ 组件和深浅色切换。
- **链接**：[github.com/liangqianxing/aurora-ui](https://github.com/liangqianxing/aurora-ui)

### 02 · Indie Radar
- **类型**：Newsletter + 数据面板
- **技术**：Next.js 15 + Supabase + Cloudflare Workers
- **亮点**：追踪独立开发者的产品上新，自动抓取商城 / Product Hunt / 少数派的热门条目。
- **链接**：[indeieradar.com](https://example.com)

### 03 · Motion Studio
- **类型**：微交互动效模版
- **技术**：After Effects + LottieFiles + Web Component
- **亮点**：把复杂的 AE 动效转换成可控参数，并在 Web 端实时预览。
- **链接**：[motion.liangkx.com](https://example.com)
